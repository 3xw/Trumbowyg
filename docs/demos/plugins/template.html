Template plugin for Trumbowyg

This plugin allows you to add a template-dropdown.

The code

    
        $('#editor')
        .trumbowyg({
            btns: ['template'],
            plugins: {
                templates: [
                    {
                        name: 'Template 1',
                        html: '<p>I am a template!</p>'
                    },
                    {
                        name: 'Template 2',
                        html: '<p>I am a different template!</p>'
                    }
                ]
            }
        });
